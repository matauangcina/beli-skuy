### Guidelines
* Create logo with **Adobe Photoshop**
* Create **gallery page** blueprint with **Adobe Photoshop** (make sure no layers are merged)
* Use **external CSS**
* No layouting with **table**
* Website is responsive for **desktops**, **tablets**, and **mobile devices**
* Apply media screen rule if main screen is below **1000px**

### Tasks
* Gallery page
  - Contain **header**, **navigation**, **content**, and **footer**
  - Image slider using **JQuery** showing latest discounts or events
  - Item cards divided into **All**, **Shoes**, **Pants**, and **Clothes**
* Register page
  - Contain **username**, **email**, **password**, **confirm password**, **date of birth**, **radio gender**, and **checkbox**
  - Use 5 kinds of validation using **Javascript** without using regex
* Chat page
* Cart page
  - Provide at least **5 items** in cart
* Profile page
  - Display user's basic information
